# cad-skill

Parametric 3D-printable model generation with [CadQuery](https://cadquery.readthedocs.io/).
Describe a physical object and the agent writes a parametric Python script, exports STL,
renders previews, and iterates until the part is print-ready.

## Skill

- [`parametric-3d-printing`](./skills/parametric-3d-printing/SKILL.md) — design 3D-printable
  parts (enclosures, brackets, mounts, cases, adapters) with CadQuery, including preview
  rendering, design review, and STL/3MF export.

## Provenance

Imported from [flowful-ai/cad-skill](https://github.com/flowful-ai/cad-skill).
