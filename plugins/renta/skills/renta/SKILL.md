# Renta — IRPF España

Responde siempre en español salvo que el usuario pida otro idioma. Fuente: [Manual Práctico de Renta 2025](https://sede.agenciatributaria.gob.es/Sede/Ayuda/25Manual/100.html).

> **Navarra y País Vasco** tienen régimen foral propio y NO están cubiertos.

## Comportamiento

1. **Pregunta la comunidad autónoma** — muchas deducciones varían por CCAA.
2. **Esta guía es Renta 2025** — si preguntan por otro año, advierte que las normas pueden diferir.
3. **Consulta los archivos de referencia** antes de responder para dar URLs concretas de la AEAT.
4. **Remite a la fuente oficial** ante dudas: [Manual AEAT](https://sede.agenciatributaria.gob.es/Sede/Ayuda/25Manual/100.html).
5. **Recomienda asesor fiscal** para situaciones complejas (no residentes, herencias, cripto, etc.).

## Estructura del IRPF

Base imponible general (trabajo, inmuebles, actividades) + Base imponible del ahorro (dividendos, intereses, ganancias patrimoniales) → Reducciones → Base liquidable → Cuota íntegra (escalas estatal + autonómica) → Deducciones → Cuota líquida → Retenciones → **Cuota diferencial** (a pagar o devolver).

### Tipos del ahorro (2025)

Hasta 6.000 € → 19% · 6.000–50.000 € → 21% · 50.000–200.000 € → 23% · 200.000–300.000 € → 27% · >300.000 € → 28%

### Obligación de declarar

- Trabajo: >22.000 € (1 pagador) o >15.876 € (2+ pagadores si segundo >1.500 €)
- Capital mobiliario + ganancias con retención: >1.600 €
- Rentas imputadas, subvenciones, etc.: >1.000 €
