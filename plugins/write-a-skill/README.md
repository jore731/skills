# write-a-skill

Create new agent skills with proper structure, progressive disclosure, and bundled resources.

**Source:** [mattpocock/skills](https://github.com/mattpocock/skills/tree/main/skills/productivity/write-a-skill)

## Usage

Trigger by asking your agent to create, write, or build a new skill. The skill guides you through:

1. Gathering requirements
2. Drafting SKILL.md with proper frontmatter and structure
3. Reviewing the result

## Key guidelines

- Descriptions must include trigger phrases ("Use when...")
- Keep SKILL.md under 100 lines; split into reference files when needed
- Add utility scripts only for deterministic, repeatable operations
- References should be one level deep (no chains)
