# DevSpace Plugin

Skills for working with [DevSpace](https://devspace.sh) — an open-source, client-only dev tool for Kubernetes (CNCF Sandbox, Apache 2.0).

## Skills

- **devspace** — Installation, CLI usage, `devspace.yaml` configuration, dev containers, deployments, pipelines, and debugging patterns.

## When to use

- Project has a `devspace.yaml` file
- User mentions DevSpace, dev containers, or Kubernetes dev environments
- Setting up hot-reload / file-sync / port-forwarding for Kubernetes workloads
- Migrating from Okteto, Tilt, or Skaffold

## Install

```shell
/plugin install devspace@skills
```
